SLRMss-package {SLRMss}R Documentation

Symmetric Linear Regression Models for small samples

Description

A function to obtain ordinary and modified statistics for symmetrical linear regression models with small samples. The supported ordinary statistics include Wald,score, likelihood ratio and gradient and the modified statistics include score, likelihood ratio and gradient.
